# ai-test-hello-world

A simple project containing "Hello World" programs in multiple programming languages.

## Project Structure

```
ai-test-hello-world/
├── README.md
├── javascript/
│   └── main.js
└── python/
    └── hello.py
```

## Hello World Javascript

To run the JavaScript Hello World program:

### Prerequisites
- Node.js installed on your system

### Execution
```bash
node javascript/main.js
```

## Hello World Python

To run the Python Hello World program:

### Prerequisites
- Python installed on your system

### Execution
```bash
python python/hello.py
```
or
```bash
python3 python/hello.py
```
