# ai-test-hello-world

A simple project demonstrating "Hello World" programs in JavaScript and Python, organized in language-specific directories.

## Project Structure

```
ai-test-hello-world/
├── README.md
├── javascript/
│   └── main.js
└── python/
    └── hello.py
```

## Hello World Javascript

### Prerequisites
- Node.js installed on your system

### How to run
```bash
node javascript/main.js
```

## Hello World Python

### Prerequisites
- Python installed on your system

### How to run
```bash
python python/hello.py
```

or

```bash
python3 python/hello.py
```
