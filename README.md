# ai-test-hello-world

A simple project demonstrating "Hello World" programs in different programming languages.

## Project Structure

```
ai-test-hello-world/
├── README.md
├── javascript/
│   └── main.js
└── python/
    └── hello.py
```

## Prerequisites

- **Node.js** - For running JavaScript files
- **Python** - For running Python files

## Hello World Javascript

To run the JavaScript Hello World program:

```bash
cd javascript
node main.js
```

Output:
```
Hello World
```

## Hello World Python

To run the Python Hello World program:

```bash
cd python
python hello.py
```

Output:
```
Hello World
```
