# CLAUDE.md

## Project Purpose

This project is for **testing purposes only**. It is used to validate an agent-driven ticket workflow where tickets are written by a human and completed by AI agents.

## Workflow Context

- Tickets are written by a human and assigned to AI agents for completion.
- Most folders and files in this repository are plain, simple scripts that exist solely to verify the workflow functions correctly.
- Do not read deeper meaning into the code structure — it is intentionally minimal.

## Guiding Principle for Agents

**Keep all implementations simple.**

- Prefer a short, direct script over an elaborate architecture.
- Do not introduce unnecessary abstractions, frameworks, or patterns.
- If a task can be done in 10 lines, do not write 100.
- Avoid over-engineering. The goal is to confirm the workflow works, not to build production-grade software.
